## 概要

各地方における潮位に関する情報を発表

**2026年5月下旬頃に平文の範疇で運用の整理あり、2028年以降に配信終了**

**資料名** <br/>
地方潮位情報

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VMCJ51

**/Report/Control/Title** <br/>
地方潮位情報

**WebSocket圧縮形式** <br/>
gzip
