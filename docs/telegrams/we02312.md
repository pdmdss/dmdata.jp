## 概要

全国における潮位に関する実況・予測を発表

**2026年5月下旬頃に配信開始**

**資料名** <br/>
全般気象解説情報（潮位）

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VMCJ53

**/Report/Control/Title** <br/>
全般気象解説情報（潮位）

**WebSocket圧縮形式** <br/>
gzip

### コード・パラメータ

全国・地方予報区等は、[気象庁防災情報XMLフォーマット](http://xml.kishou.go.jp/tec_material.html) コード表 AreaForecast.xls に記載があります。

気象情報／府県予報区・細分区域等（府県予報区等、一次細分区域等、市町村等をまとめた地域）は、気象庁防災情報XMLフォーマット コード表 AreaInformationCity-AreaForecastLocalM.xls に記載があります。

国際地点番号は、気象庁防災情報XMLフォーマット コード表 WmoObservingStations.xls に記載があります。

アメダス地点番号は、気象庁防災情報XMLフォーマット コード表 PointAmedas.xlsx に記載があります。

潮位観測地点番号は、気象庁防災情報XMLフォーマット コード表 PointTidalLevel.xls に記載があります。

他機関観測地点番号は、気象庁防災情報XMLフォーマット コード表 ObservingPointByOthers.xlsx に記載があります。

高潮予報区間は、気象庁防災情報XMLフォーマット コード表 AreaCoastForecast.xlsx に記載があります。
