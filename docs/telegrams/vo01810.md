## 概要
噴火の有無にかかわらず、噴火警戒レベル2以上の火山を対象として、降灰量及び小さな噴石の予測を定期的に発表

**資料名** <br/>
 降灰予報（定時）
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VFVO53
 
**/Report/Control/Title** <br/>
 降灰予報（定時）

**WebSocket圧縮形式** <br/>
 gzip
