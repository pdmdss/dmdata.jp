## 概要
海上火山現象に関する海上警報・予報の発表、切替及び解除について、対象火山、対象海域、警戒事項等を発表

**資料名** <br/>
 火山現象に関する海上警報・予報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VFSVii (ii=50-61)
 
**/Report/Control/Title** <br/>
 火山現象に関する海上警報・海上予報

**WebSocket圧縮形式** <br/>
 gzip
