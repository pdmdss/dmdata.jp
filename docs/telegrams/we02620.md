## 概要
台風の発生や、日本付近に位置する台風・熱帯低気圧の位置情報を発表

**資料名** <br/>
 全般台風情報(位置、発生情報)
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VPTI51

**/Report/Control/Title** <br/>
 全般台風情報（定型）
 
**WebSocket圧縮形式** <br/>
 gzip
