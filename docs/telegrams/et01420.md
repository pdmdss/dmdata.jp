## 概要
南海トラフ沿いで異常な現象が観測され、その現象が南海トラフ沿いの大規模な地震と関連するかどうか調査を開始・解説・終了した場合等に発表

**資料名** <br/>
 南海トラフ地震関連解説情報（定例以外の解説情報）
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VYSE51
 
**/Report/Control/Title** <br/>
 南海トラフ地震関連解説情報

**WebSocket圧縮形式** <br/>
 gzip
