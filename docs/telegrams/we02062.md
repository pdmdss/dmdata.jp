## 概要

土砂災害に関する警戒レベル情報（レベル５土砂災害特別警報、レベル４土砂災害危険警報、レベル３土砂災害警報、レベル２土砂災害注意報）について府県予報区ごとに発表

**2026年5月下旬頃に配信開始**

**資料名** <br/>
土砂災害に関する警戒レベル情報

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPWW56

**/Report/Control/Title** <br/>
気象警報・注意報（Ｒ０６）（土砂）

**WebSocket圧縮形式** <br/>
gzip
