## 概要

竜巻発生の恐れのある地域を1時間限り有効な情報を発表

**2026年5月下旬頃に、Head/Tileを「竜巻注意情報」から「気象防災速報（竜巻注意）」または「気象防災速報（竜巻目撃）」に変更、内容に変更ない**

**資料名** <br/>
竜巻注意情報

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPHW50

**/Report/Control/Title** <br/>
竜巻注意情報

**WebSocket圧縮形式** <br/>
gzip

### その他

発表された情報は約1時間有効であり、続報の発表をもって継続となります。
失効時刻は、/Report/Head/ValidDateTimeを必ず参照してください。
