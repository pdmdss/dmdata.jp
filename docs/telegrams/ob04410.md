## 概要

気象庁が全国6地点に設置している沿岸波浪観測計の毎正時の観測データを、沿岸波浪観測報として配信。

**2026年03月25日14時で配信終了**

**資料名** <br/>
沿岸波浪観測報

**配信形式** <br/>
国内気象通報式 JM304 沿岸波浪観測通報式

**データ種類コード** <br/>
SWJP60

**WebSocket圧縮形式** <br/>
なし
