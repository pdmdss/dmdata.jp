## 概要

土砂災害の発生の恐れがある又は、おそれが少なくなった場合に発表

**2026年05月28日に土砂災害のレベル4の発表・解除に関する場合のみ運用に変更、2028年頃に終了予定**

**資料名** <br/>

**配信形式** <br/>

**データ種類コード** <br/>
VXWW50

**/Report/Control/Title** <br/>
土砂災害警戒情報

**WebSocket圧縮形式** <br/>
gzip

### 体系整理後の運用について

この電文は体系整理後（2026年05月28日13時以降）、移行措置電文となり発表の運用が変わります。

気象等の経過措置電文（VPWW54）の発表を受け、経過措置電文の土砂災害警戒情報（VXWW50）が発表されます。
そのため、土砂災害に関する警戒レベル情報（VPWW56）で発表した内容が数分程度遅延して配信される場合があります。

体系整理と同時に、土砂災害警戒情報で使用していた地域と気象警報等で使用する地域が統一されますが、
経過措置電文（VXWW50）では体系整理前の土砂災害警戒情報の発表地域名・地域コードにより発表されます。
