## 概要

府県予報区に対する気象防災速報（線状降水帯発生、記録雨、顕著な大雪、短文形式の気象情報等。潮位に関する情報は除く）

**2026年5月下旬頃に配信開始**

**資料名** <br/>
気象防災速報

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VPBS50

**/Report/Control/Title** <br/>
府県気象防災速報

**WebSocket圧縮形式** <br/>
gzip
