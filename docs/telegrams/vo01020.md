## 概要

**資料名** <br/>

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VZVO40

**/Report/Control/Title** <br/>
火山に関するお知らせ

**WebSocket圧縮形式** <br/>
gzip
