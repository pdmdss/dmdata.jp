## 概要

中規模・大規模な噴火後、20～30分程度で詳細な降灰予報を発表。

**資料名** <br/>
降灰予報（詳細）

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VFVO55

**/Report/Control/Title** <br/>
降灰予報（詳細）

**WebSocket圧縮形式** <br/>
gzip
