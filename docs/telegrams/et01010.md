## 概要
地震・津波の試験・訓練配信のお知らせ、自治体震度データの入電停止等のお知らせ、その他を発表

**資料名** <br/>
 地震・津波に関するお知らせ
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VZSE40
 
**/Report/Control/Title** <br/>
 地震・津波に関するお知らせ

**WebSocket圧縮形式** <br/>
 gzip
