## 概要
噴火警報・噴火予報の発表・切替及び解除について、対象火山・噴火警戒レベル、火山の状況、対象市町村等を発表

**資料名** <br/>
 噴火警報・噴火予報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VFVO50
 
**/Report/Control/Title** <br/>
 噴火警報・予報

**WebSocket圧縮形式** <br/>
 gzip
