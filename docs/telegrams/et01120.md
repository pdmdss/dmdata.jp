title: 津波情報
---

## 概要
各地の満潮時刻、津波到達予想時刻に関する情報及び地上観測点における津波観測に関する情報を発表

**資料名** <br/>
 津波情報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VTSE51
 
**/Report/Control/Title** <br/>
 津波情報a

**WebSocket圧縮形式** <br/>
 gzip
