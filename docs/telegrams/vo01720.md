## 概要
対象火山・火山の状況、地震活動・微動発生状況等を発表

**資料名** <br/>
 火山の状況に関する解説情報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VFVO51
 
**/Report/Control/Title** <br/>
 火山の状況に関する火山解説情報

**WebSocket圧縮形式** <br/>
 gzip
