## 概要

初めて噴火した、またはを噴火警戒レベルを引き上げる検討を必要と考えられるような火山現象を観測した場合に速報を発表

**資料名** <br/>
噴火速報

**配信形式** <br/>
気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
VFVO56

**/Report/Control/Title** <br/>
噴火速報

**WebSocket圧縮形式** <br/>
gzip
