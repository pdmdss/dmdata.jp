## 概要

東経100°～180°、赤道～北緯60°で囲まれた北西太平洋海域を対象として、全般海上警報を発表。

「VPZU50 全般海上警報（定時）（Ｈ２９）」では、風と濃霧に関する警報が発表されていますが、新たに船体着氷のおそれがあると予想した場合に、
予想される海域に対して着氷警報を発表します。また、24時間以内に予想される風と波浪に関する予報を海域毎に提供します。

**2026年7月頃より配信開始**

**資料名** <br/>
全般海上警報（定時）（Ｒ０８）

**配信形式** <br/>

**データ種類コード** <br/>
VPZU54

**/Report/Control/Title** <br/>
全般海上警報（定時）（Ｒ０８）

**WebSocket圧縮形式** <br/>
gzip
