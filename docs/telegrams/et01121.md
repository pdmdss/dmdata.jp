## 概要
沖合の観測点における津波観測に関する情報を発表

**資料名** <br/>
 津波情報
 
**配信形式** <br/>
 気象庁防災情報XMLフォーマット

**データ種類コード** <br/>
 VTSE52
 
**/Report/Control/Title** <br/>
 沖合の津波観測に関する情報

**WebSocket圧縮形式** <br/>
 gzip
