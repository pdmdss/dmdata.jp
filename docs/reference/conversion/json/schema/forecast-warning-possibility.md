#### 早期注意情報（明後日までの警報級の可能性）(VPFD61)の場合

週間天気予報の発表単位の区域毎に、各要素の警報級の現象になる可能性を記載します。

| 階層    | フィールド      | 出現条件 | 説明                                                                                                   | 
|-------|------------|------|------------------------------------------------------------------------------------------------------|
| 1.    | region     |      | **Object**<br/> 警報級の可能性の区域予報                                                                         |
| 1._1. | type       |      | **String**<br/> `区域予報` で固定                                                                           |
| 1._2. | timeSeries |      | **Array&lt;Object&gt;**<br/> 予報対象地域の時系列で表現した予想を記載する（配列の要素は1個固定） [#1. 2. timeSeries](#1-2-timeseries) |

### 1. 2. timeSeries

| 階層       | フィールド       | 出現条件 | 説明                                                                                                           | 
|----------|-------------|------|--------------------------------------------------------------------------------------------------------------|
| 1._2._1. | timeDefines |      | **Array&lt;Object&gt;**<br/> 全ての予想期間を示すとともに、対応する要素の個々の時刻定義を記載する [#1. 2. 1. timeDefine](#1-2-1-timedefine)    |
| 1._2._2. | items       |      | **Array&lt;Object&gt;**<br/> 大雨、土砂災害、雪、風（風雪）、波、潮位について警報級の現象になる可能性の内容と発表地域を記載する [#1. 2. 2. item](#1-2-2-item) |

### 1. 2. 1. timeDefine

気象要素の時系列で参照番号として用いるための時刻IDと、基点時刻、対象期間、時刻定義の内容を記載します。

| 階層           | フィールド    | 出現条件        | 説明                                                             |
|--------------|----------|-------------|----------------------------------------------------------------|
| 1._2._1._1.  | timeId   |             | **String&lt;Integer&gt;**<br/> 時刻ID                            |
| 1._2._1._2.  | dateTime |             | **ISO8601Time**<br/> 予想する基準日時                                  |
| 1._2._1._3.  | duration |             | **String**<br/> 基準日時からの予想時間の幅を、 ISO8601 の Time intervals 形式で表す |
| 1._2._1._4.? | name     | VPFD61の場合のみ | **String**<br/> 時刻定義の内容                                        |

* 早期注意情報（明後日までの警報級の可能性）(VPFD61)の時刻定義

| 5時予報 | 11時予報 | 17時予報 | name (対象時刻) | dateTime            | duration |
|:----:|:-----:|:-----:|-------------|---------------------|----------|
|  〇   |       |       | 今日０６時から１２時  | yyyy-mm-XXT06:00:00 | PT6H     |
|  〇   |   〇   |       | 今日１２時から１８時  | yyyy-mm-XXT12:00:00 | PT6H     |
|  〇   |   〇   |   〇   | 今日１８時から２４時  | yyyy-mm-XXT18:00:00 | PT6H     |
|  〇   |   〇   |   〇   | 明日００時から０６時  | yyyy-mm-YYT00:00:00 | PT6H     |
|  〇   |   〇   |   〇   | 明日０６時から１２時  | yyyy-mm-YYT06:00:00 | PT6H     |
|  〇   |   〇   |   〇   | 明日１２時から１８時  | yyyy-mm-YYT12:00:00 | PT6H     |
|  〇   |   〇   |   〇   | 明日１８時から２４時  | yyyy-mm-YYT18:00:00 | PT6H     |
|  〇   |   〇   |   〇   | 明後日００時から１２時 | yyyy-mm-ZZT00:00:00 | PT12H    |
|  〇   |   〇   |   〇   | 明後日１２時から２４時 | yyyy-mm-ZZT12:00:00 | PT12H    |

yyyy-mm-XX は今日の、yyyy-mm-YY は明日の、yyyy-mm-ZZ は明後日の年月日の dateTime 型の意味

### 1. 2. 2. item

大雨、土砂災害、雪、風（風雪）、波、潮位について警報級の現象になる可能性の内容と発表地域を記載します。

| 階層          | フィールド | 出現条件 | 説明                                                                                     |
|-------------|-------|------|----------------------------------------------------------------------------------------|
| 1._2._2._1. | code  |      | **String&lt;Integer&gt;**<br/> 予報対象地域コード                                               |
| 1._2._2._2. | name  |      | **String**<br/> 予報対象地域名                                                                |
| 1._2._2._3. | kinds |      | **Array&lt;Object&gt;**<br/> 警報級の現象になる可能性の要素ごとに記載する [#1. 3. 2. 3. kind](#1-2-2-3-kind) |

#### 1. 2. 2. 3. kind

`波の警報級の可能性`、`潮位の警報級の可能性` は、予報区域に警報基準等がない場合は、要素ごと省略します。

| 階層             | フィールド      | 出現条件 | 説明                                      |
|----------------|------------|------|-----------------------------------------|
| 1._2._2._3._1. | properties |      | **Array&lt;Object&gt;**<br/> 配列の要素は1個固定 |

##### 1. 2. 2. 3. 1. properties

| 階層                 | フィールド                     | 出現条件      | 説明                                                                                                                                         |
|--------------------|---------------------------|-----------|--------------------------------------------------------------------------------------------------------------------------------------------|
| 1._2._2._3._1._1.  | type                      |           | **String**<br/> 気象要素の種類を表し、`大雨の警報級の可能性`、`土砂災害の警報級の可能性`、`雪の警報級の可能性`、`風（風雪）の警報級の可能性`、`波の警報級の可能性`、`潮位の警報級の可能性` を記載する                          |
| 1._2._2._3._1._2.  | possibilityRankOfWarnings |           | **Array&lt;Object&gt;**<br/> 警報級の可能性を時系列で記載する [#1. 2. 2. 3. 1. 2. possibilityRankOfWarning](#1-2-2-3-1-2-possibilityRankOfWarning)         |
| 1._2._2._3._1._3.? | texts                     | 必要に応じ出現する | **Array&lt;Object&gt;**<br/> 警報級の可能性及び対象期間の概要を文字列で記載し、同一予報区内ではtypeにかかわらず同一の文字列とする。配列の要素は1個固定  [#1. 2. 2. 3. 1. 3. text](#1-2-2-3-1-3-text) |

###### 1. 2. 2. 3. 1. 2. possibilityRankOfWarning

| 階層                    | フィールド     | 出現条件  | 説明                                                                                                                   |
|-----------------------|-----------|-------|----------------------------------------------------------------------------------------------------------------------|
| 1._2._2._3._1._2._1.  | refId     |       | **String&lt;Integer&gt;**<br/> 時刻ID                                                                                  |
| 1._2._2._3._1._2._2.  | type      |       | **String**<br/> 警報級の可能性の種類を表し、`大雨の警報級の可能性`、`土砂災害の警報級の可能性`、`雪の警報級の可能性`、`風（風雪）の警報級の可能性`、`波の警報級の可能性`、`潮位の警報級の可能性` を記載する |
| 1._2._2._3._1._2._3.  | value     |       | **String\|Null**<br/> 警報級の可能性を `高`、`中`、`なし` で記載する<br/>警報級の現象が発生する可能性が低い場合には **Null** を記載する                           |
| 1._2._2._3._1._2._4.? | condition | 場合による | **String**<br/> 警報級の現象が発生する可能性が低い場合に `値なし` と記載する                                                                     |

| value (階級値) | 解説                              |
|:-----------:|---------------------------------|
|      高      | 警報級の現象が発生する可能性が高いことを示す          |
|      中      | 警報級の現象が発生する可能性があることを示す          |
|     なし      | 季節的に現象が発生しないことを示す               |
|  **Null**   | 上記には当てはまらない（警報級の現象が発生する可能性が低い）。 |
