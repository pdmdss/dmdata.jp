## 概要

線状降水帯発生、線状降水帯直前予測、記録的短時間大雨、短時間大雪については、府県予報区ごとに「気象防災速報 (VPBS50)」で報じます。
※その他（大雨、大雪の気象防災速報等）の気象防災速報は現時点では運用されません。

台風（熱低、上陸通過情報を含む）、線状降水帯予測（半日前）、大雨等の気象情報、寒気、冬型、発達する低気圧等の気象情報、高い潮位による気象情報については、「全般気象解説情報 (VPZJ51)」、「地方気象解説情報 (VPCJ51)」、「府県気象解説情報 (VPFJ51)」でそれぞれ報じます。

大潮、高い潮位、副振動、異常潮位については、「全般気象解説情報（潮位） (VMCJ53)」、「地方気象解説情報（潮位） (VMCJ55)]」、「府県気象解説情報（潮位） (VMCJ56)」でそれぞれ報じます。

なお、「気象防災速報（潮位） (VPBS51)」については、運用開始時期未定です。

### 1. informationTag

本情報が対象とする気象現象や台風、熱帯低気圧等について、その「キーワード」を情報タグとして記載します。

| 階層     | フィールド          | 出現条件  | 説明                                                                                          |
|--------|----------------|-------|---------------------------------------------------------------------------------------------|
| 1._1.  | tagNames       |       | **Array&lt;String&gt;**<br/> 情報タグを記載する<br/>一部を省略する場合がある<br/>気象防災速報(VPBS50,51)の場合、配列の中は1個で固定 |
| 1._2.? | typhoonNumbers | 状況による | **Array&lt;String&gt;**<br/> 一連の現象が台風に関連する場合に、4桁の台風番号と接頭辞に`T`を付けたものを記載する                    |
| 1._3.? | tcNumbers      | 状況による | **Array&lt;String&gt;**<br/> 一連の現象が熱帯低気圧（台風を含む）に関連する場合に、4桁の熱帯低気圧番号と接頭辞に`TC`を付けたものを記載する      |
| 1._4.  | areas          |       | **Array&lt;Object&gt;**<br/> 本情報が対象となる区域を記載する [#1. 4. area](#1-4-area)                      |

### 1. 4. area

| 階層       | フィールド    | 出現条件 | 説明                                                                    |
|----------|----------|------|-----------------------------------------------------------------------|
| 1._4._1. | code     |      | **String&lt;Integer&gt;**<br/> 気象情報／府県予報区・細分区域等又は、全国・地方予報区等のコード       |
| 1._4._2. | codeType |      | **String**<br/> コード種別名を記載する<br/>取りうる値は、`気象情報／府県予報区・細分区域等`、`全国・地方予報区等` |
| 1._4._3. | name     |      | **String**<br/> 区域名                                                   |

情報の種別と対象となる区域の種別の関係は[別表 対象区域コード](#対象区域コード)のとおり。

通常の気象解説情報等の運用において、情報の内容に対して最小限の地域となるような地域の絞り込みは原則行われません。
例えば、波浪に関する情報において、内陸の細分区を除外するような運用は原則として行いません。
ただし、一部の気象防災速報では適切に絞り込みを行う場合があります。

### 3. observation

なお、「観測データ形式」は気象防災速報用の観測値の提供形式で、「観測実況形式」は観測実況値の単一時刻の値や期間内最大値等統計値などを提供するものです。

| 階層     | フィールド      | 出現条件                           | 説明                                                                                              |
|--------|------------|--------------------------------|-------------------------------------------------------------------------------------------------|

## この電文で取り扱うコード類

全国・地方予報区等は、気象庁防災情報XMLフォーマット コード表 AreaForecast.xls に記載があります。

気象情報／府県予報区・細分区域等（府県予報区等、一次細分区域等、市町村等をまとめた地域）は、気象庁防災情報XMLフォーマット コード表
AreaInformationCity-AreaForecastLocalM.xls に記載があります。

国際地点番号は、気象庁防災情報XMLフォーマット コード表 WmoObservingStations.xls に記載があります。

アメダス地点番号は、気象庁防災情報XMLフォーマット コード表 PointAmedas.xlsx に記載があります。

潮位観測地点番号は、気象庁防災情報XMLフォーマット コード表 PointTidalLevel.xls に記載があります。

他機関観測地点番号は、気象庁防災情報XMLフォーマット コード表 ObservingPointByOthers.xlsx に記載があります。

高潮予報区間は、気象庁防災情報XMLフォーマット コード表 AreaCoastForecast.xlsx に記載があります。
